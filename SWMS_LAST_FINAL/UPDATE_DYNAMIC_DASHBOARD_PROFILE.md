# Dynamic Dashboard & Employee Profile Update

## Added
- Dynamic month calendar with Previous, Today and Next navigation.
- Google Calendar events when the employee account is connected.
- Local task deadlines and approved leave merged into the same calendar.
- Dynamic employee statistics and task-based performance card.
- Real notification dropdown and latest notification panel.
- Employee profile photo upload (PNG/JPG/JPEG/WEBP, maximum 5 MB).
- Editable phone, address and emergency contact details.
- Protected employment fields: employee ID, department, position and joined date.
- Profile photos stored in `static/uploads/profile/` and persisted through existing storage/MySQL state.

## Run
```bash
pip install -r requirements.txt
python app.py
```
